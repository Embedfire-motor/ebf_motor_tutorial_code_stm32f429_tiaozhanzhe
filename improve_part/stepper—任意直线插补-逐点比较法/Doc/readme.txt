【*】 程序简介 
-工程名称：任意直线插补-逐点比较法
-实验平台: 野火STM32 F429 开发板
-MDK版本：5.30
-HAL库版本：1.7.6

【 ！】功能简介：
使用STM32控制两轴步进电机以起始位置为坐标原点，在任意象限内做直线插补运动
学习目的：学会基于逐点比较法的直线增量插补基本原理。

【 ！】实验操作：
下载本程序，复位开发板即可。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

X轴：
步进电机驱动器的连线接到开发板的步进电机接口1，共阳方式

	PUL+ <---> 3V3
  PUL- <---> PE9
	DIR+ <---> 3V3
	DIR- <---> PE7
  ENA+ <---> 3V3
  ENA- <---> PD0

Y轴：
步进电机驱动器的连线接到开发板的步进电机接口2，共阳方式

	PUL+ <---> 3V3
  PUL- <---> PE11
	DIR+ <---> 3V3
	DIR- <---> PE12
  ENA+ <---> 3V3
  ENA- <---> PE10
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 180MHz
-APB2 = 90MHz 
-APB1 = 45MHz

C.浮点运算单元：
  不使用
