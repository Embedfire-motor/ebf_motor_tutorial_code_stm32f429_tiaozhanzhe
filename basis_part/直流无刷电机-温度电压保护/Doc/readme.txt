【*】 程序简介 
-工程名称：直流无刷电机-温度电压保护
-实验平台: 野火STM32 F429 开发板
-MDK版本：5.30
-ST固件库版本：1.7.0


【 ！】功能简介：
使用STM32的通用定时器定时，LED1在SPWM波形控制下呈现呼吸灯效果。
学习目的：学会使用通用定时器，注意区分三种不同种类的定时器。


【 ！】实验操作：
下载本程序，复位开发板即可。

【 ！！】注意事项：
无
/*********************************************************************************************/

【*】 引脚分配

PWM：

	TIM2 CH1<--->PA15<--->舵机信号线
	
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 180MHz
-APB2 = 90MHz 
-APB1 = 45MHz

C.浮点运算单元：
  不使用
