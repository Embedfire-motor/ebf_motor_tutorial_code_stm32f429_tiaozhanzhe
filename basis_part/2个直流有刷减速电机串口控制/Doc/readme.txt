【*】 程序简介 
-工程名称：2个直流有刷减速电机串口控制
-实验平台: 野火STM32 F429 开发板
-MDK版本：5.30
-HAL库版本：V1.7.6

【 ！】功能简介：
通过串口发送指令,设置电机方向和速度.
指令示例:
d1 0
v1 1000
电机1方向正转,速度设置为1000,注意指令结尾需要换行,或在串口助手中勾选发送新行

【 ！】实验操作：

电机1接线:
电机驱动板 5V\GND 		<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 	<----> 开发板 PE9\PE11
电机驱动板 SD 				<----> 开发板 PG12

电机2接线:
电机驱动板 5V\GND 		<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 	<----> 开发板 PC6\PC7
电机驱动板 SD 				<----> 开发板 PE6

下载本程序，复位开发板即可。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

定时器TIM1输出PWM信号:
两路PWM输出到PE9\PE11上,连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PE9\PE11

电机使能引脚:
电机驱动板 SD <----> 开发板 PG12

定时器TIM8输出PWM信号:
两路PWM输出到PC6\PC7上,连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PC6\PC7

电机使能引脚:
电机驱动板 SD <----> 开发板 PE6
